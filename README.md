# 💰 Pricing Engine

The pricing section is powered using a configuration-based pricing matrix.

Supported:

* 🇮🇳 INR
* 🇺🇸 USD
* 🇪🇺 EUR

Billing:

* Monthly
* Annual

Annual pricing is calculated dynamically using:

```
Annual = Monthly × 12 × 0.8
```

This eliminates hardcoded values and allows scalable pricing management.

# 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd aetherdata_ai_landing
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
